这两天想看，比如说郑渊洁短篇集，发现微信读书很多书都下架了

也看了几篇关于此事（“纸电之争”）的新闻解读。微信读书等电子书平台等变现方式：会员订阅制、单品销售制、在会员订阅的基础上单品销售。平台主推的当然是订阅制，出版社想要的是单品销售制。*当然如此，订阅制对出版社来说则完全是个黑箱，单品销售制则是“按件计费”，不会扯皮。*

微信读书的内容就是两部分，网文和严肃文学。网文好做，因为网文市场起点一家独大，而起点也是腾讯旗下的，那为什么微信读书不主打网文呢？ 因为微信读书只是起点的二道贩子，与QQ阅读什么的一样，一个销售渠道而已。严肃文学就不同了，各种出版社，国企有中信出版社、人民文学出版社、商务印书馆、人民邮电出版社、中华书局等国家一级出版社就近百家，民企有磨铁、果麦之类的，还有巴黎评论之类的外刊，说白了出版物市场都不能说是群雄逐鹿了，只能说是纷纷扰扰、乱七八糟。现在全球范围内还真没有一家能把这些出版社绑定起来的电子书平台。*当然，我对此是悲观的，这种费力不讨好的事情，真的是完完全全的鸡肋，估计腾讯什么时候会放弃这部分业务也不一定。*

另外，*需要注意的是即使是单品销售制，平台上的电子书通常比纸质书还贵，就很离谱了。*

***最后，不要订阅会员，也不要买电子书，都是蠢炸了。最好的仍然是从网上找PDF资源。***

- 不要自己下载PDF再导入了，大部分导入的PDF都无法转成电子书，展示效果很差，文字都看不清，微信读书上资源是专门定制的电子版，插图质量就很好

可以简单理解为，体验卡和赠币是对位的

```markdown
微信读书在55断更节后调整过用户协议，赠币不能购买了，如果之前一直续费会员的话，用的是调整前的协议不变，赠币可以看网文，老人老办法，新人新办法。不过微信读书也在调整网文上架速度和范围，很多非头部网文只能在起点看到，微信读书是不会上了，毕竟微信读书上的购买，不会给作者分成，起点的购买才可以
```

最后，我想探究一下这种别扭感的来源。主要是两方面

- 这些书并不是什么稀缺资源，网上都能下载到PDF，更何况微信读书的资源并不全，很多书都搜不到，并做不到一站式的搜书和阅读体验，无法带来更好的体验。
